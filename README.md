# SQL Learning Portfolio

This repository tracks my SQL learning as I move from a research scientist
role into data-focused work.

I learn new tools the same way I learned science: understand the basics,
test ideas with small examples, and check results carefully before moving on.
The work here reflects that approach.

The repository is organised by week. Each week includes:
- short notes explaining key ideas in plain language
- SQL queries that focus on one concept at a time
- selected coursework submissions
- brief reflections on what worked, what didn’t, and what to practise next

The SQL here is not production code. It’s learning code.
The goal is to show clear thinking, careful logic, and growing confidence
with core SQL concepts such as filtering, operator precedence, and sorting.

I use this repository to document progress and working style as I build
solid SQL foundations.
